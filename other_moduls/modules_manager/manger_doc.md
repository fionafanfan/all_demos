# 文档管理工具

## zeal 
Zeal 是一款免费的离线文档软件，能够让开发者更加便捷地查阅api文档。目前该软件有 windows、mac 以及 linux 版本，内部有超过 200 个文档，涵盖了几乎所有程序开发用到的库、框架以及语言，是一款十分实用的软件。

zeal链接： 
Zeal官网：[zealdocs.org/]()
Zeal下载：[zealdocs.org/download.html]()

Docsets 集合： https://zealusercontributions.vercel.app/docsets
